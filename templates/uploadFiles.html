Quick Cracking Guide: (Using Hashcat)

 ** Hashcat removes duplicates by default.
1. With the PwdDump file, extract the following columns:
  • username (1st column)
  • NTLM hash (4th column)
    • # cut -d: -f1,4 pwdumpFile.txt > output_user_ntlm.txt
  • the content of the 'output_user_ntlm.txt' file should be:
    • user1:NTLM_value
    user2:NTLM_value
    user...:..
2. In Hashcat:
  • clear the 'hashcat.pot' content
  • execute hashcat with the '--username' argument against the 'output_user_ntlm.txt' file
  • the resulting file or the 'hashcat.pot' content would be:
    • NTLM_value:password
    NTLM_value:password
    NTLM...:
3. Upload the files
  • the tool will combine the two files into:
    • User:NTLM_value:password
